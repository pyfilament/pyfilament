Follow these steps to install Pyfilament, write your first task, and explore the key decorator options.

<Steps>
1. Install Pyfilament
    Install `pyfilament` from PyPI. Pyfilament requires Python 3.14 or higher.

    ```bash uv
    uv add pyfilament
    ```

    ```bash pip
    pip install pyfilament
    ```

2. Write your first task
    Import `task` from `filament` and decorate any `async def` function. Calling the function creates a managed task run that executes immediately.

    ```python
    from filament import task
    from asyncio import run

    @task
    async def greet(name: str) -> str:
        return f"Hello, {name}!"

    run(greet("World"))
    ```

    `@task` used without parentheses applies default configuration: one attempt, no timeout, no caching. Calling `greet("World")` returns the task run result directly — you do not need to unwrap it separately.

3. Add retries and a timeout
    Pass configuration arguments to `@task(...)` to make the function resilient. The example below retries up to three times, waits one second between attempts, and cancels any run that exceeds ten seconds.

    ```python
    from filament import task
    from asyncio import run

    @task(tries=3, timeout=10.0, delay=1.0)
    async def fetch_data(url: str) -> dict:
        # your async HTTP code here
        ...

    run(fetch_data("https://api.example.com/data"))
    ```

    | Argument | Type | Description |
    | --- | --- | --- |
    | `tries` | `int` | Total number of attempts before the run fails. Default: `1`. |
    | `timeout` | `float \| None` | Seconds before the run is cancelled. Default: `None` (no limit). |
    | `delay` | `float` | Seconds to wait between retry attempts. Default: `0`. |
    | `backoff_base` | `float` | Exponential backoff multiplier applied to `delay`. Default: `2`. |

4. Use get_logger inside a task
    Call `get_logger()` from inside any task to get a logger that is automatically scoped to the current task run. Logs are captured in the task run record and visible in the observability UI.

    ```python
    from filament import task, get_logger
    from asyncio import run

    @task
    async def process(item: str) -> str:
        logger = get_logger()
        logger.info(f"Processing {item}")
        return item.upper()

    run(process("hello"))
    ```

    When called outside a task, `get_logger()` returns a standard logger scoped to the calling module and function — so it is always safe to use.
